# AnalyX - 学术统计软件

## 产品介绍

AnalyX 是一款专业学术科研统计软件，具有以下特点：

- **桌面离线软件**：本地运行，数据安全
- **现代风格 GUI**：基于 PyQt6，现代简洁的用户界面
- **功能全面**：覆盖 SPSS 标准版所有核心功能
- **速度超快**：基于 NumPy、SciPy，性能优异
- **支持安装包分发**：提供 Windows 安装包
- **可直接开发落地**：完整的项目代码和构建配置

## 核心功能

### 数据准备
- 数据导入：Excel (.xlsx/.xls)、CSV、TXT、SPSS .sav 格式
- 数据清洗：缺失值处理、异常值检测、重编码
- 变量计算、派生变量

### 描述统计
- 频数表、描述统计（均值、中位数、标准差、偏度、峰度）
- 百分位数、四分位数、直方图、正态性检验

### 均值比较与检验
- 单样本 t 检验
- 独立样本 t 检验
- 配对样本 t 检验
- 单因素方差分析 ANOVA

### 非参数检验
- 卡方检验
- Mann-Whitney U 检验
- Kruskal-Wallis 检验
- Friedman 检验

### 相关与回归
- Pearson、Spearman 相关
- 简单线性回归
- 多元线性回归

### 量表与问卷专用
- 信度分析：Cronbach's α
- 效度分析
- 因子分析
- 聚类分析

### 可视化
- 条形图、折线图、散点图、箱线图
- 直方图、Q-Q 图、P-P 图
- 热力图、ROC 曲线
- 学术规范图表，可直接用于论文

### 导出
- APA 格式统计表
- 图表高清导出（PNG/TIFF）
- SPSS .sav 兼容导出

## 技术架构

### 技术栈
- **GUI 框架**：PyQt6（现代美观的跨平台界面）
- **计算内核**：NumPy、SciPy（高性能科学计算）
- **数据处理**：Pandas（强大的数据处理能力）
- **图表库**：Matplotlib、Seaborn（专业的学术图表）
- **数据读取**：openpyxl、pyreadstat（支持 Excel 和 SPSS 格式）

### 界面结构
- 顶部菜单：文件、数据、分析、图表、导出、设置
- 左侧：功能导航（分类清晰）
- 中央：数据表格视图
- 右侧：结果输出 + 图表
- 底部：状态条、日志、进度
- 深色 / 浅色模式切换

## 快速开始

### 环境要求
- Python 3.8 或更高版本
- Windows 10/11、macOS 或 Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python src/main.py
```

### 构建 Windows 安装包

1. 首先使用 PyInstaller 打包成单文件：
```bash
python build.py
```

2. 然后使用 Inno Setup 制作安装程序：
```
打开 installer.iss 文件，使用 Inno Setup Compiler 编译
```

## 用户指南

### 导入数据
1. 点击菜单：文件 → 导入
2. 选择要导入的文件类型（CSV、Excel、SPSS）
3. 选择数据文件
4. 数据会自动加载到表格中

### 进行统计分析
1. 在左侧导航栏选择分析类型
2. 或从菜单中选择：分析 → [分析类型]
3. 在弹出的对话框中选择变量
4. 点击"计算"按钮
5. 分析结果会显示在"分析结果"标签页

### 绘制图表
1. 点击菜单：图表 → [图表类型]
2. 选择要绘制的变量
3. 点击"绘制"按钮
4. 图表会显示在"图表"标签页

### 导出结果
1. 点击菜单：文件 → 导出
2. 选择导出格式
3. 选择保存位置
